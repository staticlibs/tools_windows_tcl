pg_lo_close

pg_lo_close — Close a large object

Synopsis

pg_lo_close conn descriptor

Description

pg_lo_close closes a large object.

Arguments

conn

The handle of the connection.

descriptor

A descriptor for the large object from pg_lo_open.

Return Value

Nothing. A Tcl error will be thrown if an error occurs.

Notes

All large object manipulation must take place within an SQL transaction block.

This command uses or emulates the PostgreSQL libpq function lo_close.

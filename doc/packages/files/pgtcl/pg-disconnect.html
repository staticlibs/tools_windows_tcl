pg_disconnect

pg_disconnect — Close a connection to the server

Synopsis

pg_disconnect conn

Description

pg_disconnect closes a connection to the PostgreSQL server.

Arguments

conn

The handle of the connection to be closed.

Return Value

None. A Tcl error will be thrown if an error occurs.

Notes

This command uses or emulates the PostgreSQL libpq function PQfinish.

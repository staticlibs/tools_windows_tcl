Chapter 4. pgtcl Command Reference

Abstract

This chapter contains the reference information for all the pgtcl commands. The commands are grouped into functional categories. (An alphabetical listing of commands can be found in Table 1.1, “pgtcl Commands”.)

Note: This chapter follows the Tcl documentation convention for indicating optional parameters with question marks ?like this...? rather than the typical notation with square brackets.

4.1. Database Connection Control Commands

pg_connect — Open a connection to the server
pg_disconnect — Close a connection to the server
pg_conndefaults — Get connection options and their defaults

Abstract

The commands in this section are used to connect to and disconnect from the database.
